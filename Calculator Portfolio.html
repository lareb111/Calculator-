📒 How It Works — Notes

3 Languages Used

  • HTML — buttons & screen structure
  • CSS — design, colors, layout
  • JavaScript — calculation logic

Key JS Concepts

  • let — variable store karna
  • eval() — string ko math mein convert karna
  • onclick — button press detect karna
  • getElementById — HTML element pakadna
  • try/catch — error handle karna

Logic Flow

Button press → string mein add hota hai → = press hone pe eval() se answer aata hai → screen pe show hota hai

Why Not Python?

Python backend hai — browser pe directly nahi chalta. Website calculator ke liye JavaScript best hai kyunki yeh browser mein directly run hoti hai — koi server nahi chahiye.
